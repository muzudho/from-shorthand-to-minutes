# from-shorthand-to-minutes

速記したものを議事録の形式に変換

速記の例：  

```plaintext
むずでょ    日曜日に速記から議事録を起こさな。
            大変だなあ

きふわらべ  お父ん、働けよ
```

形式的には以下の通り：  

```plaintext
氏名　ホワイトスペース　発言  
行頭タブ　ホワイトスペース　発言  
    ※３行目以降は、２行目の形式を繰り返し

これが複数人分続く
```

これを以下のデータ形式に変換：  

```json
[
    {"氏名", "発言１行目", "発言２行目"}
    {"氏名", "発言１行目", "発言２行目"}
]
```

このデータができれば、あとは好きなようにフォーマットを定めて出力
